# 员工论坛系统 (Discuz! X3.5二次开发)

## 项目概述
- **项目名称**: 企业员工论坛
- **类型**: 内部社区系统
- **核心功能**: 论坛发帖、评论、敏感词审核、访问控制
- **目标用户**: 企业内部员工 + 企业微信用户

## 访问控制策略
| 来源 | IP范围 | 认证方式 |
|------|--------|----------|
| 办公网 | 172.20.0.0/16 | 用户名密码 |
| 本机 | 127.0.0.1 | 用户名密码 |
| 企业微信工作台 | OAuth2登录 | 企业微信OAuth2 |
| 其他 | - | 403拒绝 |
